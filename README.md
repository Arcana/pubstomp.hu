pubstomp.hu
===========

The code behind http://pubstomp.hu - powered by Flask

# Requirements
* Python 2.7
* [Bower](http://bower.io/)
* [SASS](http://sass-lang.com/)
* [Virtualenv](https://virtualenv.pypa.io/en/latest/)

# Installing

Install dependencies
`pip install -r requirements.txt`
`bower install`
